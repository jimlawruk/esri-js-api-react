# ESRI JS API Starter Map using React and Vite

A starter template for an ESRI ArcGIS JS API map using React and Vite.

[!https://raw.githubusercontent.com/jimlawruk/esri-js-api-react/main/screenshot.png]

## Setup

```
  npm install
```

## Run Locally

```
  npm run dev
```

## Build

```
  npm run build
```
